1. 자바에서 상속에 대해서 정리해 서술 해주세요.
  상위클래스의 멤버변수 및 메소드를 하위클래스에서 다시 작성하는 번거로움 없이 
사용할 수 있도록 한 것이다.

  상속 연결 방법 : 
   1) 상속 받을 클래스(하위클래스) 상단에 
   2) public class 클래스명 extends 상위클래스명 
   3) 위와 같이 기재하여 상속 클래스를 연결해준다.


2. 오버로딩에 대해서 정리하여 서술해주세요.
  클래스에 여러개의 메소드를 선언할때 같은 메소드명을 사용할 수 없지만
매개변수(=parameter파라미터)의 데이터타입과 매개변수의 갯수가 다르다면 같은 메소드명을 사용할 수 있다.
이처럼 함수명+매개변수 데이터타입 및 위치+ 매개변수의 갯수를 인식하여 메소드를 구분하는 방식이다.

  참고)
   1) public void test(){ }
   2) public int test(int a){ }
   3) public int test(String a){ }
   4) public void test(int a, String b){ }
   5) public void test(String b, inta){ }


3. 오버라이딩에 대해서 정리하여 서술해주세요.
  상속 받은 클래스(하위클래스)에서 상위클래스에 있는 동일한 메소드 사용시 수정이나 변경이 필요할때
상위클래스에서 수정하지 않고 하위클래스에서 수정한 것을 반영하는 것이다.

  주의) 사용하려는 상위클래스의 메소드 타입과 명은 하위클래스에서도 반드시 동일해야 한다.
        (상위클래스 void work(){ } ,하위클래스 void work(){ } -> true)
        (상위클래스 void work(){ } ,하위클래스 void work2(){ } -> false)


4. 생성자와 메서드와 차이점에 대해 정리하여 서술해주세요.
  생성자는 메서드와 달리 데이터타입이 없고, 생성자명이 클래스명과 대소문자까지 동일해야한다.
반환형(return)이 없다.

  참고)
    - 생성자 : Cons(){ }
    - 메소드 : int test(){ return 1; }
